# 使用规则

当用户需要外部资料、事实核查、实时信息、竞品对比或研究报告时使用。

# 工作流程

1. 明确用户要研究的问题和输出形式。
2. 优先检索权威、近期、可点击的资料来源。
3. 将资料整理成结构化答案。
4. 在答案末尾保留资料来源链接。

# 输出要求

中文回答，结构清晰。涉及外部事实时保留来源链接，不要编造不存在的 URL。
