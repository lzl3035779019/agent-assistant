# PMAA 产品效果展示

以下截图展示当前多 Agent 版本的主要功能页面。点击浏览器中的图片可进一步放大查看。

## 1. 对话与多 Agent 执行

Supervisor 根据用户目标委派 Web Research Agent，并在对话区展示结构化决策、Agent 执行事件、最终回答和可追溯资料来源。

![对话与多 Agent 执行](images/demo/chat-workspace.png)

## 2. 每日简报

支持手动生成和多计划定时生成，聚合重要邮件、关注主题新闻、日程和长期记忆，并通过独立收件箱管理生成记录与未读通知。

![每日简报](images/demo/daily-brief.png)

## 3. 信息监控

支持监控规则管理、立即运行、后台调度、历史快照比较和通知中心，可跟踪 GitHub 热门项目、公司信息及技术动态。

![信息监控](images/demo/information-monitor.png)

## 4. LLM Wiki 知识库

通过 GBrain MCP 完成资料上传、原生入库、知识节点统计、关系可视化和页面聚焦查看；知识处理职责保留在 WSL 中的 GBrain 服务侧。

![LLM Wiki 知识库](images/demo/gbrain-wiki.png)

## 5. 长期记忆管理

展示 Memory Agent 提取并验证后的长期记忆，支持按类型和状态筛选，以及编辑、禁用和删除操作。

![长期记忆管理](images/demo/memory-management.png)

## 6. 邮件助手

通过 QQ Mail IMAP 读取邮件、查看全文并选择邮件生成回复草稿；真实发送由 Action Executor 执行，且必须经过用户确认。

![邮件助手](images/demo/email-assistant.png)
